## Getting Started

1. Clone the project
```
git clone https://github.com/ozankaraca/issue-reporter.git
```
2. Create a branch for your changes from `main`

```
git checkout -b YOUR_BRANCH_NAME main
```
3. Make Your Changes and Commit Them
4. Create a pull request to merge them to main
